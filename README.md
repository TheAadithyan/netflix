# Netflix - Clone

This project is a simplified front end clone of Netflix. 
It was created with React and CSS (Grid and Flexbox). It uses The TMDB API.

[Click Here For Preview](https://netflix-clone-b01d6.web.app/)

## Tech Stack

- Webpack 
- Axios
- React
- Flexbox
